O Website de Gestão de Recibos do Colégio Rhulany é uma aplicação web desenvolvida para facilitar a emissão, 
visualização, armazenamento e impressão de recibos escolares pela secretaria. O sistema é simples, rápido, intuitivo e totalmente responsivo,
funcionando bem em computadores e tablets.
A aplicação permite emitir recibos para alunos da 1ª à 6ª classe, com número gerado automaticamente de forma sequencial (sem repetição),
data automática e layout profissional pronto para impressão em formato A4. Cada página contém dois recibos (parte superior e inferior), 
permitindo o corte ao meio após a impressão.
O sistema possibilita inserir dados como nome do aluno, classe, número do aluno (opcional), nome do encarregado (opcional),
descrição do pagamento (propina, matrícula, uniforme, material escolar ou outro), valor pago, valor por extenso em Meticais
(gerado automaticamente) e forma de pagamento (Dinheiro, M-Pesa, e-Mola ou Transferência bancária).

Tecnologias usadas
Frontend: React + TypeScript
Design: Tailwind CSS
Backend: Node.js
Base de Dados: PostgreSQL (Neon)

Gestão de Pacotes: npm(node Package Manager)


Como Executar o Projeto
Instalar as dependências:
npm install
Executar em ambiente de desenvolvimento:
npm run dev
Gerar build para produção:
npm run build
